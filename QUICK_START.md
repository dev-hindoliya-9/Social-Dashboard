# Quick Start Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   - Navigate to `http://localhost:5173`
   - The dashboard should load automatically

## What You'll See

- **Header**: Search bar, notifications, settings, and user profile
- **Sidebar**: Navigation menu with dashboard options
- **Stats Cards**: Key metrics (Followers, Reach, Engagement, Comments)
- **Charts**: 
  - Engagement overview (line chart)
  - Audience growth (area chart)
- **Platform Widgets**: Analytics for Twitter, Facebook, Instagram, LinkedIn, YouTube
- **Top Posts**: Best performing content
- **Recent Activity**: Live feed of interactions
