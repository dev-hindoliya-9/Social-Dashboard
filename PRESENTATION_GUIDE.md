## Troubleshooting

**Problem**: Dashboard nahi khulta
**Solution**: 
- `start.bat` double-click karein
- Ya terminal mein `npm run dev` run karein

**Problem**: Error dikh raha hai
**Solution**: 
- Pehle `install.bat` run karein (dependencies install karne ke liye)
